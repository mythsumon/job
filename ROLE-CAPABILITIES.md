## 🌐 공통 기능 (모든 사용자)

### 비로그인 사용자 (Guest)도 사용 가능

#### 1. 홈페이지 (`/`)
- ✅ 채용공고 검색
- ✅ 추천 채용공고 조회
- ✅ 프리미엄 채용공고 조회
- ✅ 기업 목록 조회
- ✅ 플랫폼 통계 확인

- ✅ 채용공고 상세보기
- ✅ 채용공고 저장 (로그인 필요)

#### 3. 기업 탐색 (`/user/companies`)
- ✅ 기업 목록 조회
- ✅ 기업 검색 및 필터링
- ✅ 기업 상세 정보 확인
- ✅ 기업의 채용공고 목록 확인
- ✅ 기업 리뷰 확인

#### 4. 기타 공통 페이지
- ✅ 커리어 정보 (`/user/career`)
- ✅ 피드 (`/user/feed`)
- ✅ 회원가입 (`/register`)
- ✅ 로그인 (`/login`)

## 🔐 접근 제어 규칙

### 1. 인증 필요 (Protected Routes)
- 모든 `/user/*`, `/company/*`, `/admin/*` 경로는 로그인 필요
- 비로그인 사용자는 로그인 페이지로 리다이렉트

### 2. 역할 기반 접근 제어 (Role-Based Access Control)
- **RoleGuard**: 특정 사용자 유형만 접근 가능
- **ProtectedPage**: 로그인 필요
- **AuthGuard**: 인증 상태 확인

### 3. 채팅 규칙
- ⚠️ **중요**: 구인자만 채팅을 시작할 수 있음
- 구직자는 구인자가 시작한 채팅에만 참여 가능
- 한 채용공고당 한 구직자와 하나의 채팅방만 존재

### 4. 지원 규칙
- 구직자만 채용공고에 지원 가능
- 한 채용공고당 한 번만 지원 가능
- 지원 취소 가능
