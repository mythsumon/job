# Job Mongolia Platform - 웹사이트 개요

## 🌟 플랫폼 소개

**Job Mongolia**는 몽골의 구인구직 및 채용공고 플랫폼입니다. 구직자와 구인자를 연결하여 효율적인 채용 프로세스를 제공하는 현대적이고 혁신적인 플랫폼입니다.

---

## 🎯 플랫폼 목적

### 주요 목표
1. **구직자와 구인자의 효율적인 연결**
   - 채용공고 탐색 및 지원 프로세스 간소화
   - 실시간 채팅을 통한 직접 소통

2. **몽골 시장 특화**
   - 몽골 ID 시스템 지원
   - 몽골어, 한국어, 영어 다국어 지원
   - 몽골 지역 특화 기능

3. **현대적인 채용 경험 제공**
   - AI 기반 인재 추천
   - 실시간 채팅
   - 종합적인 채용 관리 도구

---

## 👥 사용자 유형

### 1. 구직자 (Candidate)
- **대상**: 일자리를 찾는 사람들
- **주요 기능**: 채용공고 탐색, 지원, 이력서 관리
- **목적**: 이상적인 직장 찾기

### 2. 구인자 (Employer)
- **대상**: 인재를 찾는 기업 및 회사
- **주요 기능**: 채용공고 작성, 지원자 관리, 채용 프로세스 관리
- **목적**: 적합한 인재 채용

### 3. 관리자 (Admin)
- **대상**: 플랫폼 운영자
- **주요 기능**: 플랫폼 관리, 사용자/기업 관리, 시스템 모니터링
- **목적**: 플랫폼의 원활한 운영

---

## 🏠 주요 페이지 및 기능

### 공통 페이지 (모든 사용자 접근 가능)

#### 1. 홈페이지 (`/`)
- **목적**: 플랫폼의 첫인상 및 주요 정보 제공
- **주요 콘텐츠**:
  - 추천 채용공고
  - 프리미엄 채용공고
  - 인기 기업
  - 플랫폼 통계
  - 빠른 검색 기능

#### 2. 채용공고 목록 (`/user/jobs`)
- **목적**: 모든 채용공고 탐색
- **주요 기능**:
  - 검색 및 필터링
  - 위치, 산업, 경력, 급여별 필터
  - 원격근무 옵션
  - 정렬 옵션

#### 3. 채용공고 상세 (`/user/jobs/:id`)
- **목적**: 특정 채용공고의 상세 정보 확인
- **주요 정보**:
  - 채용공고 설명
  - 요구사항
  - 급여 정보
  - 기업 정보
  - 지원하기 버튼

#### 4. 기업 목록 (`/user/companies`)
- **목적**: 등록된 기업 탐색
- **주요 기능**:
  - 기업 검색
  - 산업별 필터링
  - 기업 규모별 필터링

#### 5. 기업 상세 (`/user/companies/:id`)
- **목적**: 기업의 상세 정보 확인
- **주요 정보**:
  - 기업 소개
  - 기업 문화
  - 복리후생
  - 해당 기업의 채용공고 목록
  - 기업 리뷰

## 👤 구직자 전용 기능

### 대시보드 (`/user/home`)
- **목적**: 구직자의 개인화된 홈 화면
- **주요 콘텐츠**:
  - 맞춤형 채용공고 추천
  - 저장한 채용공고
  - 지원 현황 요약
  - 빠른 액션

### 이력서 관리 (`/user/resumes`)
- **목적**: 전문적인 이력서 작성 및 관리
- **주요 기능**:
  - 여러 이력서 작성 및 관리
  - 다양한 템플릿 선택
  - 이력서 미리보기
  - 기본 이력서 설정
  - 이력서 다운로드/공유

### 지원 프로세스
1. 채용공고 탐색
2. 채용공고 상세 확인
3. 이력서 선택 또는 작성
4. 지원서 제출
5. 지원 현황 확인
6. 기업과 채팅 (기업이 시작)

### 채팅 (`/user/chat`)
- **목적**: 기업과의 직접 소통
- **주요 기능**:
  - 실시간 메시지 교환
  - 채팅방 목록
  - 읽지 않은 메시지 알림
  - 채팅방 관리

## 🏢 구인자 전용 기능

### 기업 대시보드 (`/company/dashboard`)
- **목적**: 채용 활동의 전체적인 현황 파악
- **주요 콘텐츠**:
  - 채용 통계
  - 최근 지원자
  - 활성 채용공고 현황
  - 빠른 액션 메뉴

### 채용공고 관리 (`/company/jobs`)
- **목적**: 채용공고 작성 및 관리
- **주요 기능**:
  - 새 채용공고 작성
  - 채용공고 수정/삭제
  - 채용공고 상태 관리
  - 프리미엄/추천 공고 설정

### 지원자 관리 (`/company/applications`)
- **목적**: 지원자 검토 및 평가
- **주요 기능**:
  - 지원자 목록 확인
  - 지원서 및 이력서 검토
  - 지원 상태 변경
  - 지원자 필터링

### 채용 파이프라인 (`/company/pipeline`)
- **목적**: 채용 프로세스의 단계별 관리
- **주요 기능**:
  - 지원자 단계별 이동
  - 면접 일정 관리
  - 채용 단계별 통계

### 인재 검색 (`/company/talents`)
- **목적**: 적극적인 인재 발굴
- **주요 기능**:
  - 인재 검색 및 필터링
  - 인재 프로필 확인
  - 채팅 시작 (구인자만 가능)
  - AI 기반 인재 추천

### 채팅 (`/company/chat`)
- **목적**: 구직자와의 직접 소통
- **주요 기능**:
  - 채팅 시작 (구인자만 가능)
  - 실시간 메시지 교환
  - 채팅방 관리

## 📝 요약

**Job Mongolia**는 몽골의 구인구직 시장을 위한 종합적인 플랫폼입니다. 구직자와 구인자를 효율적으로 연결하고, 현대적인 채용 도구를 제공하여 성공적인 채용을 돕습니다. 실시간 채팅, AI 기반 추천, 종합적인 채용 관리 기능을 통해 사용자에게 최고의 경험을 제공합니다.
