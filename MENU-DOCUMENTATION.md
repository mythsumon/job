# Job Mongolia Platform - Menu Documentation

## 개요
이 문서는 Job Mongolia 플랫폼의 모든 메뉴와 각 메뉴의 기능을 상세히 설명합니다.

---

## 1. 공개 메뉴 (Public Menus) - 모든 사용자 접근 가능

### 1.1 홈페이지 (`/`)
- **경로**: `/` 또는 `/user/home`
- **접근 권한**: 모든 사용자 (로그인 여부 무관)
- **기능**:
  - 추천 채용공고 조회
  - 프리미엄 채용공고 조회
  - 기업 목록 조회
  - 플랫폼 통계 정보 확인
  - 채용공고 검색
- **주요 액션**:
  - 채용공고 상세보기로 이동
  - 기업 상세보기로 이동
  - 로그인/회원가입

### 1.2 채용공고 목록 (`/user/jobs`)
- **경로**: `/user/jobs`
- **접근 권한**: 모든 사용자
- **기능**:
  - 채용공고 검색 (키워드, 위치, 산업 등)
  - 고급 필터링:
    - 위치 (지역)
    - 산업 분야
    - 경력 수준
    - 급여 범위
    - 고용 형태 (정규직, 계약직, 인턴 등)
    - 원격근무 여부
  - 채용공고 카드 클릭 → 상세보기
  - 채용공고 저장 (로그인 사용자만)
- **정렬 옵션**:
  - 최신순
  - 급여순
  - 관련도순

### 1.3 채용공고 상세 (`/user/jobs/:id`)
- **경로**: `/user/jobs/:id`
- **접근 권한**: 모든 사용자
- **기능**:
  - 채용공고 상세 정보 조회:
    - 직무 설명
    - 자격 요건
    - 급여 정보
    - 근무 조건
    - 복리후생
  - 기업 정보 확인
  - 지원하기 버튼 (로그인한 구직자만)
  - 저장하기 버튼 (로그인한 구직자만)
  - 공유하기 기능

### 1.4 기업 목록 (`/user/companies`)
- **경로**: `/user/companies`
- **접근 권한**: 모든 사용자
- **기능**:
  - 기업 검색 및 필터링:
    - 산업 분야
    - 회사 규모
    - 위치
  - 기업 카드 클릭 → 상세보기
  - 기업 리뷰 확인

### 1.5 기업 상세 (`/user/companies/:id`)
- **경로**: `/user/companies/:id`
- **접근 권한**: 모든 사용자
- **기능**:
  - 기업 정보 조회:
    - 회사 소개
    - 회사 문화
    - 복리후생
    - 연락처 정보
  - 해당 기업의 채용공고 목록
  - 기업 리뷰 및 평점
  - 기업 팔로우 (로그인 사용자만)

### 1.6 커리어 가이드 (`/user/career`)
- **경로**: `/user/career`
- **접근 권한**: 모든 사용자
- **기능**:
  - 커리어 관련 정보 및 리소스
  - 이력서 작성 가이드
  - 면접 준비 팁
  - 직업 탐색 도구
  - 커리어 개발 자료

### 1.7 커뮤니티 피드 (`/user/feed`)
- **경로**: `/user/feed`
- **접근 권한**: 모든 사용자
- **기능**:
  - 채용 관련 뉴스 및 피드
  - 커뮤니티 게시글
  - 업계 트렌드 정보
  - 사용자 게시글 작성 (로그인 사용자만)
  - 좋아요 및 댓글 기능

### 1.8 로그인 (`/login`)
- **경로**: `/login`
- **접근 권한**: 비로그인 사용자
- **기능**:
  - 이메일/비밀번호 로그인
  - 로그인 성공 시 사용자 유형에 따라 리다이렉트:
    - 구직자 → `/user/home`
    - 구인자 → `/company/dashboard`
    - 관리자 → `/admin/dashboard`

### 1.9 회원가입 (`/register`)
- **경로**: `/register`
- **접근 권한**: 비로그인 사용자
- **기능**:
  - 계정 유형 선택 (구직자/구인자)
  - 회원가입 폼 작성
  - 이메일 인증
  - 몽골 ID 검증 (몽골인인 경우)
  - 회원가입 완료 및 자동 로그인

### 1.10 요금제 소개 (`/pricing`)
- **경로**: `/pricing`
- **접근 권한**: 모든 사용자 (로그인 여부 무관)
- **기능**:
  - 요금제 비교 테이블:
    - Basic (무료)
    - Professional (99,000 MNT/월)
    - Enterprise (299,000 MNT/월)
  - 각 플랜의 기능 목록
  - 가격 정보
  - "지금 시작하기" 버튼
  - FAQ 섹션
  - 고객 후기/평점
- **다음 액션**:
  - 로그인한 구인자: 구독 신청으로 이동
  - 비로그인 사용자: 로그인 페이지로 이동

## 2. 구직자 (Candidate) 전용 메뉴

### 2.1 프로필 (`/user/profile`)
- **경로**: `/user/profile`
- **접근 권한**: 로그인한 구직자
- **기능**:
  - 프로필 정보 수정:
    - 이름, 이메일, 전화번호
    - 위치, 생년월일
    - 프로필 사진 업로드
    - 자기소개
    - 기술 스택
    - 경력 정보
    - 학력 정보
  - 프로필 공개 설정
  - 프로필 미리보기

### 2.2 이력서 관리 (`/user/resumes`)
- **경로**: `/user/resumes`
- **접근 권한**: 로그인한 구직자만
- **기능**:
  - 이력서 목록 조회
  - 새 이력서 작성:
    - 개인 정보
    - 경력 사항
    - 학력 사항
    - 기술 및 능력
    - 자격증 및 수상 내역
    - 자기소개서
  - 이력서 수정 및 삭제
  - 기본 이력서 설정
  - 이력서 미리보기
  - 이력서 다운로드 (PDF)
  - 이력서 템플릿 선택

### 2.3 지원 현황 (`/user/applications`)
- **경로**: `/user/applications`
- **접근 권한**: 로그인한 구직자만
- **기능**:
  - 지원한 채용공고 목록 조회
  - 지원 상태 확인:
    - 지원 완료
    - 서류 검토 중
    - 면접 대기
    - 면접 완료
    - 합격/불합격
  - 지원서 상세보기
  - 지원 취소 (상태에 따라)
  - 지원 통계:
    - 총 지원 수
    - 진행 중인 지원
    - 합격률

### 2.4 저장한 채용공고 (`/user/saved-jobs`)
- **경로**: `/user/saved-jobs`
- **접근 권한**: 로그인한 구직자만
- **기능**:
  - 저장한 채용공고 목록 조회
  - 저장한 채용공고 삭제
  - 저장한 채용공고 필터링
  - 지원하기 (저장한 채용공고에서 직접)

### 2.5 채팅 (`/user/chat`)
- **경로**: `/user/chat`
- **접근 권한**: 로그인한 구직자
- **기능**:
  - 채팅방 목록 조회
  - 기업과의 1:1 채팅:
    - 실시간 메시지 전송/수신
    - 파일 첨부
    - 읽음 확인
    - 타이핑 인디케이터
  - 채팅방 검색
  - 읽지 않은 메시지 알림
  - 채팅방 닫기/재개
  - 채팅 기록 조회

### 2.6 설정 (`/user/settings`)
- **경로**: `/user/settings`
- **접근 권한**: 로그인한 구직자
- **기능**:
  - 알림 설정:
    - 이메일 알림
    - 푸시 알림
    - 채용공고 추천 알림
    - 지원 상태 변경 알림
  - 개인정보 설정:
    - 프로필 공개 범위
    - 연락처 공개 설정
  - 비밀번호 변경
  - 계정 삭제
  - 언어 설정
  - 테마 설정 (다크/라이트 모드)

### 3.1 기업 대시보드 (`/company/dashboard`)
- **경로**: `/company/dashboard`
- **접근 권한**: 로그인한 구인자만
- **기능**:
  - 통계 정보:
    - 전체 채용공고 수
    - 월간 지원자 수
    - 채용공고 조회수
    - 예정된 면접 수
  - 최근 지원자 목록
  - 활성 채용공고 현황
  - 빠른 액션:
    - 새 채용공고 작성
    - 지원서 검토
    - 면접 일정 잡기
  - 채용 파이프라인 현황
  - 성과 목표 진행률
  - 최근 활동 타임라인
  - 지원 트렌드 차트
  - 채용공고 성과 분석

### 3.2 채용공고 관리 (`/company/jobs`)
- **경로**: `/company/jobs`
- **접근 권한**: 로그인한 구인자만
- **기능**:
  - 채용공고 목록 조회:
    - 활성 채용공고
    - 비활성 채용공고
    - 만료된 채용공고
    - 초안 채용공고
  - 새 채용공고 작성:
    - 직무 제목 및 설명
    - 자격 요건
    - 급여 정보
    - 근무 조건
    - 복리후생
    - 마감일 설정
    - 채용공고 카테고리
  - 채용공고 수정 및 삭제
  - 채용공고 상태 변경 (활성/비활성)
  - 채용공고 복제
  - 채용공고 통계:
    - 조회수
    - 지원자 수
    - 채용 완료 수

### 3.3 지원자 관리 (`/company/applications`)
- **경로**: `/company/applications`
- **접근 권한**: 로그인한 구인자만
- **기능**:
  - 지원자 목록 조회:
    - 전체 지원자
    - 서류 검토 중
    - 면접 대기
    - 합격/불합격
  - 지원서 상세보기:
    - 지원자 정보
    - 이력서 확인
    - 지원 동기
    - 첨부 파일 확인
  - 지원 상태 변경:
    - 서류 검토 중
    - 면접 대기
    - 면접 완료
    - 합격
    - 불합격
  - 지원자 필터링 및 검색
  - 지원자에게 채팅 시작
  - 면접 일정 잡기
  - 지원자 평가 및 메모 작성

### 3.4 채용 파이프라인 (`/company/pipeline`)
- **경로**: `/company/pipeline`
- **접근 권한**: 로그인한 구인자만
- **기능**:
  - 지원자 파이프라인 관리:

### 6.1 지원 프로세스
1. 구직자가 채용공고 상세 페이지에서 "지원하기" 클릭
2. 이력서 선택 또는 새로 작성
3. 지원서 제출
4. 구인자가 지원서 확인 (`/company/applications`)
5. 구인자가 채팅 시작 가능
6. 면접 일정 조율 (`/company/interviews`)
7. 합격/불합격 처리 (`/company/pipeline`)
8. 고용 완료 (`/company/employees`)

### 6.2 채팅 플로우
1. **채팅 시작**: 구인자만 채팅 시작 가능
2. **메시지 전송**: 실시간 WebSocket 통신
3. **채팅방 관리**: 채팅방 닫기/재개

### 6.3 채용공고 작성 플로우
1. 구인자가 `/company/jobs` 접근
2. "새 채용공고 작성" 클릭
3. 채용공고 정보 입력
4. 채용공고 저장 및 게시
5. 지원자 확인 및 관리
